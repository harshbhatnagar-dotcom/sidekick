# Sidekick 🤖

Sidekick is an AI-powered agent built with LangGraph, Groq, Gradio, and Playwright. It can search the web, browse webpages, execute Python code, manage files, query Wikipedia, and send notifications.

## Features

- 🌐 Web Search (Tavily)
- 🕸️ Webpage Browsing (Playwright)
- 📚 Wikipedia Search
- 🐍 Python Code Execution
- 📁 File Management
- 📧 Email Notifications
- 🔄 Multi-step Agent Workflow with LangGraph



## Tech Stack

- LangGraph
- LangChain
- Groq
- Gradio
- Playwright
- Tavily
- Resend
